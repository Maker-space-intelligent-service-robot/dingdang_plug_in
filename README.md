# dingdang_plug_in
叮当机器人的插件集
# 安装
```
cd /home/pi/.dingdang
git clone git@github.com:Maker-space-intelligent-service-robot/dingdang_plug_in.git contrib
pip install -r contrib/requirements.txt
```
# 升级
```
cd /home/pi/.dingdang/contrib
git pull
pip install --upgrade -r requirements.txt
```
